# Openclaw安全检查工具 - 规格文档

## 1. 项目概述

- **项目名称**: Openclaw Security Scanner
- **项目类型**: macOS 开源安全工具
- **核心功能**: 检测Macbook上的Openclaw安全风险，提供一键修复功能
- **目标用户**: Macbook用户、安全从业者、开发人员

## 2. UI/UX 规格

### 2.1 窗口结构

- **主窗口**: 单窗口应用，尺寸 900x600，最小尺寸 800x500
- **布局**: 左右分栏设计
  - 左侧: 安全风险列表 (250px宽度)
  - 右侧: 详情与操作区域

### 2.2 视觉设计 (极客风格)

**色彩方案**
- 主背景色: #0D1117 (深色背景)
- 侧边栏背景: #161B22
- 卡片背景: #21262D
- 主色调 (安全绿): #3FB950
- 警告色 (橙色): #F0883E
- 危险色 (红色): #F85149
- 文字主色: #E6EDF3
- 文字次色: #8B949E
- 边框色: #30363D

**字体**
- 主字体: SF Pro Display, -apple-system, sans-serif
- 代码字体: SF Mono, Menlo, monospace
- 标题: 24px 粗体
- 副标题: 18px 中等
- 正文: 14px 常规
- 小字: 12px

**间距系统**
- 基础间距: 8px
- 卡片内边距: 16px
- 区块间距: 24px
- 窗口边距: 20px

### 2.3 组件设计

**标题栏**
- 应用图标 + 标题 "Openclaw Security Scanner"
- 右侧: 扫描按钮 + 状态指示器

**左侧风险列表**
- 可滚动的风险项列表
- 每项包含: 风险图标 + 风险名称 + 风险等级标签
- 选中状态: 左边框高亮 (#3FB950)
- 悬停状态: 背景色变亮

**风险等级标签**
- 严重: 红色背景 (#F85149)
- 警告: 橙色背景 (#F0883E)
- 信息: 蓝色背景 (#58A6FF)

**右侧详情区域**
- 风险标题与大图标
- 风险描述
- 详细说明
- "一键修复" 按钮 (绿色主按钮)
- "查看详情" 链接

**扫描状态**
- 扫描进行中: 动画进度条 + 旋转图标
- 扫描完成: 勾选图标 + "扫描完成"
- 有风险: 警告图标 + 风险数量

**一键修复按钮**
- 悬停: 亮度提升
- 点击: 波纹效果
- 修复中: 加载动画
- 修复成功: 绿色勾选反馈

## 3. 功能规格

### 3.1 安全检查项目

1. **Openclaw安装检测**
   - 检测Openclaw应用是否安装
   - 检测Openclaw相关进程是否运行
   - 检测Openclaw系统服务状态

2. **安全风险项** (左侧列表)
   - Openclaw未安装 (信息级)
   - Openclaw运行中 - 隐私风险 (严重级)
   - Openclaw网络连接 (警告级)
   - Openclaw配置异常 (警告级)
   - Openclaw版本过旧 (信息级)

3. **一键修复功能**
   - 退出Openclaw进程
   - 卸载Openclaw应用
   - 清理残留文件
   - 重置网络配置

### 3.2 用户交互流程

1. 打开应用 → 自动开始扫描
2. 扫描完成 → 左侧显示风险列表
3. 点击风险项 → 右侧显示详情
4. 点击"一键修复" → 确认对话框 → 执行修复 → 显示结果

### 3.3 数据处理

- 本地检测，无需网络
- 使用系统API检测应用安装情况
- 使用Shell命令检测进程和服务

## 4. 技术实现

- **框架**: Electron (跨平台桌面应用)
- **前端**: HTML/CSS/JavaScript
- **后端**: Node.js + 系统命令调用
